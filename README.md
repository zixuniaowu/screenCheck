# Schedule Diff Checker - 安装测试说明

甘特图/调度表差分检测工具，用于对比页面变化并高亮显示差异。

## 一、获取插件

```bash
git clone https://github.com/zixuniaowu/screenCheck.git
```

或直接下载 ZIP：https://github.com/zixuniaowu/screenCheck/archive/refs/heads/main.zip

---

## 二、Edge 浏览器安装步骤

1. 打开 Edge 浏览器
2. 地址栏输入 `edge://extensions/` 并回车
3. 找到页面**左侧边栏底部**的 **"开发人员模式"** 开关，**打开它**
4. 打开后，页面顶部会出现新的按钮，点击 **"加载解压缩"**
5. 在弹出的文件夹选择窗口中，选择下载的 `screenCheck` 文件夹
6. 安装成功后，浏览器右上角工具栏会出现插件图标（点击拼图图标可以看到）

---

## 三、使用步骤

1. **打开目标页面**（甘特图/调度表页面）

2. **点击浏览器右上角的插件图标**，会弹出操作面板

3. **保存基准快照**：点击 `保存快照 A` 按钮
   - 会提示保存成功，显示提取了多少个 slot

4. **等待页面发生变化后**（或者滚动到其他区域后），点击 `保存快照 B`

5. **点击 "对比差异" 按钮**
   - 如果有差异，页面上会用彩色边框高亮显示变化的位置
   - 绿色 = 新增的 slot
   - 橙色 = 内容变更的 slot
   - 红色 = 被删除的 slot

6. **点击 "清除高亮"** 可以去掉页面上的标记

## 五、常见问题

**Q: 插件图标在哪里？**
A: Edge 右上角，可能被折叠在拼图图标里，点击拼图图标可以看到

**Q: 提示"提取数据失败"怎么办？**
A: 需要根据实际页面结构调整代码，请提供 F12 控制台的信息

**Q: 插件加载失败？**
A: 确保"开发人员模式"已开启，并且选择的是包含 manifest.json 的文件夹
